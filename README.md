# Unsupervised Traffic State Discovery Using Multi-Feature Vehicle Analysis from Urban CCTV Streams

[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📋 Overview

Complete research repository for unsupervised traffic state discovery using multi-feature vehicle analysis from urban CCTV streams. This pipeline automatically detects vehicles, tracks them, extracts traffic features, performs temporal aggregation, and discovers traffic states using DBSCAN clustering.
